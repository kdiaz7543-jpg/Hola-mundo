# **App Name**: Basic Display

## Core Features:

- Display Text: Display the text 'Hola Mundo' on the screen.

## Style Guidelines:

- Primary color: Soft blue (#64B5F6) to create a calm and clean aesthetic.
- Background color: Light grey (#FAFAFA) for a neutral backdrop.
- Accent color: Pale green (#B2DFDB) for interactive elements, offset by the blue primary.
- Font: 'Inter', a sans-serif font for a clean and modern look.
- Center the text on the screen for easy visibility.
